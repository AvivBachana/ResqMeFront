# ResQme React App

A React + Vite starter app for the ResQme landing experience.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- This project is scaffolded as a responsive React app.
- The current UI is built to match a modern resume builder landing page.
- If you provide Figma assets or design details, I can refine the layout to match the exact screens.
